## Installation

### Prérequis
- Node.js 20+
- npm

### Installation
```bash
npm install
```

### Variables d'environnement
Créer un fichier `.env.local` à la racine :

VITE_GITHUB_TOKEN=votre_token_github

VITE_GITHUB_REPO=Igbes-07/pokedex-s6

### Lancer le projet
```bash
npm run dev
```

### Lancer les tests
```bash
npm run test
```
